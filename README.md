# 项目概述 java481

本项目是基于 Django 的图书管理系统，融合了后端的 Python 3.8 和 Django 3.2，以及前端的 JavaScript 和 Vue 技术进行开发。

# 技术栈

## 后端

- Python 3.8
- Django 3.2
- 数据库：MySQL 5.7

## 前端

- 前端技术栈：ES6、Vue、Vuex、Vue-Router、Vue-Cli、Axios、Ant Design

## 开发工具

- PyCharm
- VSCode

## 运行环境

- Windows 10/11

# 功能模块

## 前台功能

- 首页
- 图书详情
- 用户中心

## 后台功能

- 总览
- 借阅管理
- 图书管理
- 分类管理
- 标签管理
- 评论管理
- 用户管理
- 运营管理
- 日志管理
- 系统信息

# 系统角色

系统分为前台用户角色和后台管理员角色。
